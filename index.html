CSS SYNTAX

Common Selectors

  • Type selector: p { }
  • Class selector: .btn { }
  • ID selector: #header { }
  • Descendant: .card p { }
  • Child: .nav > li { }
  • Attribute: input[type="text"] { }
  • Pseudo-classes: a:hover, input:focus
  • Specificity rules (short): inline > id > class / attribute / pseudo-class > type
