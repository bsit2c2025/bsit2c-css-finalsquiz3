WHAT IS CSS?

  • CSS = Cascading Style Sheet — language that controls presentation of HTML.
  • Separates content (HTML) from presentation (CSS).
  • The cascade, inheritance, and specificity are core behaviors.

INLINE CSS

Inline CSS refers to the method of applying CSS styles directly to a single HTML element using the style attribute within the element's opening tag.

This approach allows for specific styling of individual elements without affecting other elements on the page.

Code

<p style="color:red;">Hello</p>

Output

Hello

INTERNAL CSS

Internal CSS, also known as embedded CSS, is a method of styling HTML elements by including CSS rules directly within the section of a single HTML document.

<head>
    <style>
        body {font-family:Arial, sans-serif;}
    </style>               
</head>

EXTERNAL CSS

External CSS refers to a method of styling web pages where the CSS rules are stored in a separate file with a .css extension, distinct from the HTML document. 

<!-- index.html -->
<link rel="stylesheet" href="styles.css">

styles.css :

body { font-family: Arial, sans-serif; }

CSS SYNTAX

selector{property:value,property2:value2}

Example : 

h1 {
    color:#203040;
    font-size:36px;
    margin-bottom:0.5rem;
}

Common Selectors

  • Type selector: p { }
  • Class selector: .btn { }
  • ID selector: #header { }
  • Descendant: .card p { }
  • Child: .nav > li { }
  • Attribute: input[type="text"] { }
  • Pseudo-classes: a:hover, input:focus
  • Specificity rules (short): inline > id > class / attribute / pseudo-class > type

CSS PROPERTIES

  • Text: color, font-size, line-height, text-align
  • Box: width, height, padding, margin, border, box-sizing
  • Background: background-color, background-image, background-size
  • Display: display, visibility
  • Positioning: position, top/right/bottom/left, z-index

THE BOX MODEL

Concept: Every element is a rectangular box with (inside→out): content → padding → border → margin.

Example HTML :

<div class="box">Content</div>

CSS Example : 

.box{
    width:200px;
    padding:20px;
    border:4px solid #333;
    margin:16px;
    background:#f5f5f5;
}

Code Output :

Content
